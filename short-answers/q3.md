# Question
What is polymorphism? Show an example in a code snippet.

# Answer
Polymorphism same name method different implementation.
```js
class Car {
  constructor(make, model) {
    this.make = make;
    this.model = model;
  }
  drive() {
    console.log("Vrooom");
  }
}
class RaceCar extends Car { 
  drive() { 
    console.log("Vah... Vah...");
    super.drive();
    console.log("WHEEEEEEE!!!!");
  }
}

```
